Pseudocodigo:

# UI OBJECTS
    ## MAP
    Add Map

    Add markers de waypoints GIVEN:
        waypoints contiene localizacion, distancia, carga adicional y color de acuerdo al tiempo que a~ade a la ruta)
    
    Add layer de waypoints
        contiene color: 
            gris oscuro es carga adicional no sobre pasa la carga maxima del camion, gris casi transparente es carga adicional sobre pasa la carga maxima del 
            camion, informacion de carga adicional, tiempo a~adido a la ruta.
    
    Add marker de camion(Localizacion real-time) HOW TO RECREATE

    Add layer de camion    
    Add layer de ruta
    
    ## TRUCK CARD
    Add informacion de camion(contiene toda la informacion sobre el camion)<div>
        contiene los botones de Overview, Follow y Reset

EVENTS
    Tener un map.on('load', function(){})

    Tener un map.on('click', function(){}): //on click on waypoints

        Si la carga adicional es mayor que la capacidad. rechaza el pickup.
        Cuando uno hace click a un waypoint lo a~ade a la ruta.
        Recalcula la ruta optimizada.
        Guarda y remueve el punto seleccionado de Waypoints.
        Add punto a CurrentPoints(prompt color azul) contiene el orden para visitar(e.g Pickup #1, Pickup #2, etc).
        Actualizar ruta.
